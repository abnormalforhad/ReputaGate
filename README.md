# ReputaGate: The FairScale Reputation Portal

ReputaGate is a production-ready Solana application that leverages **FairScale** reputation infrastructure to reward high-integrity users with premium DeFi yields and exclusive access.

## üõ°Ô∏è FairScale Integration
ReputaGate integrates the FairScale API to verify wallet reputation in real-time. This data is used to:
- **Gate Access**: Only wallets with a "Silver" tier or higher can access the platform.
- **Scale Rewards**: APR bonuses are dynamically calculated based on the `fairscore` (e.g., Platinum users get +10% APR).
- **Social Proof**: Displays badges (Diamond Hands, LST Maxi) directly in the user profile.

## üõ†Ô∏è Setup Instructions

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables in a `.env` file:
   ```env
   VITE_FAIRSCALE_API_KEY=your_fairscale_api_key_here
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
