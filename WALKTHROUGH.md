# ReputaGate - The FairScale Reputation Portal

ReputaGate is a production-ready Solana application that leverages **FairScale** reputation scores to provide users with a personalized, high-integrity experience.

## Features Implemented
- **Modern Dark Glassmorphism**: A high-fidelity, moody dark aesthetic with Amber accents and premium serif typography.
- **FairScale Reputation Analysis**: Seamlessly connects to the FairScale API (with mock fallback) to fetch FairScore, Tiers, and Badges.
- **Dynamic Reward Engine**: Automatically scales DeFi APR bonuses based on the user's reputation tier.
- **Exclusive Feature Gating**: High-tier users gain access to "Elite" staking vaults and premium platform features.
- **Premium Design System**: A bespoke Vanilla CSS design featuring glassmorphism, fluid gradients, and micro-animations.

## Technical Stack
- **Frontend**: React (Vite) + TypeScript
- **Styling**: Modern Vanilla CSS
- **Infrastructure**: FairScale API (Reputation Layer)
